# Introduction to XeoContext

XeoContext is the modern home for your architecture context.

## What is it?

It unifies:
- **System Design** (Markdown + Mermaid)
- **API Specs** (OpenAPI/Swagger)
- **Event Driven Architecture** (AsyncAPI)

...into one cohesive, searchable, and AI-friendly site.

## Getting Started

1. Install Docker.
2. Run `docker run -p 3000:3000 xeost/xeocontext`.
